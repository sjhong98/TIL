## 상수

한번 저장하면 데이터값 변경 불가

상수는 대문자로 정의

```java
int i = 10;
i = 5;

final int J = 10;
J = 5   // 불가
```

### 문자형

### 실수형

### 정수형

- long (뒤에 l 붙여야 함)


## 기본형의 타입 변환

### 묵시적 형변환
크기가 더 큰 타입으로 변환할 때 사용
```java
int x = 50000;
long y = x;
```
### 명시적 형변환
크기가 더 작은 타입으로 변환할 때 사용
```java
long x = 50000;
int y = (int) x;
// int y = x -> 에러
```
